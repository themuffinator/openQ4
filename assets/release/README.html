openQ4

Release package for the open-source Quake 4 engine and game-code replacement.

This archive contains openQ4 binaries plus the staged baseoq4/ runtime payload. Quake 4 assets are not included.

Open Documentation Portal Project Overview Technical Reference

Requirements

  • A legitimate Quake 4 installation from Steam or GOG.
  • The stock q4base PK4 files available through openQ4's asset discovery or a manual fs_basepath setting.

Quick Start

  1. On Windows, either run the installer executable or extract the archive to a folder of your choice. The installer can detect an existing openQ4 install, adds a standard uninstall entry, and optionally registers openq4:// browser links.
  2. Run openQ4-client_<arch> or openQ4-client_<arch>.exe.
  3. On Steam Deck, use openQ4-steamdeck when it is included in the package.
  4. Open docs/index.html for the full offline documentation set, or use the installer-created Start Menu shortcuts on Windows.
  5. If auto-detection fails, launch with +set fs_basepath "path\\to\\Quake 4".

Package Contents

  • openQ4-client_<arch> and openQ4-ded_<arch>: engine binaries.
  • baseoq4/pak0.pk4: openQ4-owned runtime content and overrides.
  • docs/index.html: full packaged HTML documentation portal.
  • docs/docs-user/ and docs/docs-dev/: converted user and developer references.
  • VERSION.txt: packaged version metadata.
  • LICENSE: GPLv3 engine license text.

Documentation

The release archive includes a full offline HTML docs site instead of only a single readme. Start with docs/index.html, then move into the overview, build guide, technical reference, user guides, and developer notes.
